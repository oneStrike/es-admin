# 作者角色管理功能实现计划

## 功能需求

在作者管理页面中增加角色管理功能，允许管理员在弹窗中管理作者角色（添加、编辑、删除、调整状态），参考数据字典子项的实现方式。

## 实现方案

### 1. 创建角色管理组件

* 创建 `src/views/content-manager/author-manager/role.vue` 组件，参考数据字典子项的实现

* 实现角色列表展示、添加、编辑、删除、状态切换功能

### 2. 更新作者管理主页面

* 修改 `index.vue`，添加角色管理弹窗

* 添加"角色管理"按钮，点击打开角色管理弹窗

### 3. 实现角色管理逻辑

* 集成现有的角色类型API：

  * `authorRoleTypeListApi` - 获取角色列表

  * `authorRoleTypeCreateApi` - 创建角色

  * `authorRoleTypeUpdateApi` - 更新角色

  * `authorRoleTypeDeleteApi` - 删除角色

  * `authorRoleTypeChangeStatusApi` - 切换角色状态

### 4. 更新shared.ts配置

* 添加角色管理相关的表单schema、列定义等

* 更新作者表单中的角色选择，使用动态获取的角色列表

## 技术实现要点

1. **弹窗组件设计**：参考 `DictionaryItem` 组件，使用 `useVbenModal` 创建弹窗
2. **数据交互**：使用现有的角色类型API，实现完整的CRUD操作
3. **角色选择优化**：将作者表单中的硬编码角色选项改为动态获取
4. **UI一致性**：保持与数据字典子项相同的交互风格
5. **响应式设计**：确保弹窗在不同屏幕尺寸下正常显示

## 预期效果

* 作者管理页面工具栏增加"角色管理"按钮

* 点击按钮打开角色管理弹窗，包含角色列表和操作按钮

* 支持添加新角色、编辑现有角色、删除角色、切换角色状态

* 作者表单中的角色选择框使用最新的角色列表

## 实现步骤

1. 创建 `role.vue` 组件
2. 更新 `shared.ts`，添加角色管理相关配置
3. 修改 `index.vue`，集成角色管理弹窗
4. 测试角色管理功能的完整流程
5. 验证作者表单中的角色选择是否正常工作
